# 白话文翻译

南郭子綦（zǐ qí）隐居在机巧之处坐着，仰望天空而呼气，回答他的样子好像失去了配偶。颜成子游站在他前面，问道：“你为什么待在这里？身体当然可以让它像枯木一样，心灵也可以让它像死灰一样吗？如今那些隐居的人，不再像以前那些隐居的人了。”子綦回答说：“偃（yǎn），不是很好吗？你来问我这个！现在我失去了自我，你知道吗？你听到人的声音却没有听到地的声音，你听到地的声音却没有听到天的声音！”

子游说：“请问其方法。”子綦回答：“大块（kuài）呼吸，其名为风。这只是无所作动，作动的话，万窍就会愤怒地喧哗，只有风不会发出翏翏（huī huī）的声音吗？山林里畏惧美好的东西，大树百围的孔穴，像鼻子，像嘴巴，像耳朵，像枅（yù），像圈，像臼（jiù），像洼的地方，像污浊的地方；激动的，謞（mó）动的，叱（chì）喊的，吸（xī）气的，叫喊的，譹（mián）动的，宀夭（miān yāo）的，咬（yǎo）合的，前面的唱歌，随后的唱喁（yūn），泠风则小和，飘风则大和，厉风济则众窍为虚，而独不见其调调、之刀刀（dāo dāo）？”

子游说：“地的声音就是众窍已经，人类的声音就是像竹子的声音而已，敢问天的声音是什么？”子綦回答：“风吹万物不同，但让它们自我存在，都是它们自己选择，愤怒的人又算谁呢？”

大智慧的人显得闲适，小智慧的人显得犹豫不决；大言说话激烈，小言说话谨慎。人在睡眠时，魂魄交汇，醒来时，形体展开，与接触为构造，日复一日心中争斗：缦（màn）者，窖（jiào）者，密（mì）者。小的恐惧不安，大的恐惧缦缦。其发若机栝（gāo），其司是非之谓也；其留如诅盟（zǔ méng），其守胜之谓也；其杀若秋冬，以言其日消也；其溺之所为之，不可使复之也；其厌也如缄（jiān），以言其老洫（xù）也；近死之心，莫使复阳（yáng）也。喜怒哀乐，虑叹变埶（tì）心，姚佚启态；乐出虚，蒸成菌（jūn）。日夜相代于前，而无人知其所萌。已乎，已乎！早晚得到这个，是什么使它生存呢！

不是彼无我，也不是我无所取。这已经很接近了，却不知道它是如何运作的。好像有真宰（zǎi），但无法真正看到。可以依行己信（xìn），却看不见它的形态，有情而无形。百骸、九窍、六藏，全部存有其中，我又能和谁亲近呢？你们都说了吗？它有私心吗？像是所有的臣仆吗？这些臣仆不足以相互治理吗？它们相互递换君臣吗？它有真正的君主存在吗？如果寻求得到它的情感却得不到，又有什么益处，不会损害它的真理吗？一旦接受它的形成，却不忘记等待其尽头。与万物相互作用如刀剑相交，它们行尽就像驰骋一样，没有人能够停止，这不是悲哀吗！终身劳累而看不到成功，疲劳不知归宿，这难道不令人哀伤吗！人们说不死，那又有什么益处！它的形态变化，它的心灵与之相同，难道不能说是大悲哀吗！人生下来，固然像是带有芒刺一样吗？我独自有芒刺，而人们也有没有芒刺的人吗？跟随它形成心灵去模仿它，谁不曾没有师傅呢？为什么一定要知道变化而心灵自我选择呢？愚人和聪明人一样。心灵未形成就有是非，是今天适应变化而过去到了也是。因此无有为就有。有无为就有，虽然有神禹（yǔ），也不能知道，我独自又能怎么办呢！

言语不是风吹的，言者有言，它所言的东西还没有确定。果真有言吗？它从未有过言吗？它认为与鷇（qiāo）音不同，也有辨别吗？它没有辨别吗？道难道在隐蔽中有真假吗？言难道在隐蔽中有是非吗？道难道在去往中不存吗？言难道在存在中不可？道隐藏于小的成就，言隐藏于荣华。因此有儒墨之是非，因而他们所非的是他们所是，所是的则非他们所是。想要是他们所非的就是他们所非，反之亦然，那最好的办法就是明白。

万物没有不是彼的，万物没有不是是的。自彼方不见，自我知晓就知晓。因此说彼出于此，是也因彼而生。彼是方生之说，虽然如此，方生方死，方死方生；方可方不可，方不可方可；因是因非，因非因是。因此圣人不依靠，而照耀在天上，也是因彼而如此。这也是彼，彼也是这样。彼也是一是非，这也是一是非。果真有彼吗？果真无彼吗？彼是没有偶然的，称之为道枢（shū）。枢开始得到它的中心，来应对无穷。因此说它也是一无穷，非也是一无穷。因此说最好用明亮的东西来指代，指代非指，或用非指来指代指；用马来比喻马的非马，不如用非马来比喻马的非马。天地是一只手指，万物是一匹马。

可行的就是可行，不可行的就是不可行。道行进并成就，万物说它就这样。因此难道是这样吗？是这样就是如此，不是这样就是如此，没有辩论的余地；但是如果是自然的，那么自然的就如此，不然的也不然，自然如此，自然不然。万物固然有所是的，万物固然有所可的。没有东西不自然，没有东西不可自然。因此说，以“是”举起莛（tíng）与楹（yíng），厉与西施，恢恑（huī qiàn）憰怪，道通为一其部分，也就是成就；它的成就也是毁灭。所有的东西既没有成就也没有毁灭，重新通为一。只有聪明的人知道通为一，成为“是”却不用，寄托于庸俗中。庸俗的人，就是使用它；使用它的人，就是通晓它；通晓它的人，就是得到它；恰好得到的人也就几乎了。因为如此。这样做却不知道为什么如此，称之为道。费神明力去统一却不知道它们的同源，称之为朝三。什么是朝三？狙公（jū gōng）献芧（qí），说：“早晨三次，晚上四次。”众多狙人都愤怒。说：“那么早晨四次，晚上三次。”众多狙人都同意。名实并不亏欠而喜怒成为用处，也是因为如此。因此圣人用是非来调和天平，称之为两行。

古代的人，他们的知识有一定的境界。为什么没有达到呢？因为他们认为未曾有过物，已经达到了，已经尽了，无法再增加。其次认为有物了，却从未封存。其次认为有封存，却从未有是非。这是非的显著，是道亏损的原因。道亏损，是爱欲使然。果真有成就和亏损吗？果真无成就和亏损吗？有成就和亏损，所以昭氏（zhāo shì）弹鼓琴；无成就和亏损，所以昭氏不弹琴。昭文（zhāo wén）弹琴，是师旷（shī kuàng）的枝策，是惠子（huì zǐ）执梧，是三子的知识之端，他们都是最盛的时候，因此在晚年被载入末年。只有他们喜欢，以不同于他人，他们喜欢以为明亮。那些不是所明亮而明亮的，用坚白的昧终；而他们的后代用文饰来纶终，终身没有成就。如果这样能称为成就吗？即使我也成就了。如果这样不能称为成就呢？万物和我无成就。因此滑疑（huá yí）耀，是圣人所追求的。成为“是”不用而寓于庸俗，这就是用明亮来指代。

现在有言语在这里，不知道它是和“是”相类的吗？是和不是相类的吗？类和不类，相互构成类，那么与彼就没有差别了。然而，请容我尝试说一下。有开始的事物，有未曾有开始的事物，有未曾有开始的人。有有的事物，有无的事物，有未曾有有的事物，有未曾有无的事物。忽然间有无了，却不知道有无的根源哪个有哪个无。现在我有了说法，却不知道我所说的根源是否有名气，它的根源是否无名气？天下没有比秋豪（qiū háo）的末端更大的了，而大山却变得小；没有比殇子（shāng zǐ）更长寿的了，而彭祖（péng zǔ）却过早而死。天地与我同生，万物与我同一。既已成为一，又如何有言语呢？既已说成了一，又如何有无言语呢？一和言语为二，二和一为三。从此以后，巧妙的计策无法得到，更何况是普通的！所以自无适有以至于三，更何况自有适有呢！没有适应，依循如此。

道从未封存，言语从未有常，用“是”而有所界限。请说出它的界限：有左，有右，有伦，有义，有分，有辩，有竞，有争，这些就是八德。六合之外，圣人存在却不讨论；六合之内，圣人讨论却不议论。春秋经世先王之志，圣人讨论却不辩论。因此，分就是有不分；辩就是有不辩。为什么呢？圣人心怀它，众人以辩论来互相展示。因此说辩论的人，有不见的东西。大道不称，大辩不言，大仁不仁，大廉不嗛（qiǎn），大勇不忮（shì）。道显露却不是道，言辩却不到位，仁常却不成，廉清却不信，勇忮却不成。五者聚集而趋向方位。因此知道止于自己不知道的地方，达到了。谁知道不言的辩论，不知道道的道？如果有人能知道，这就是天府（tiānfǔ）。注视它却不满足，倾注它却不竭，却不知道它的来处，这就是保持光明。

因此，昔日尧帝问舜帝说：“我想砍伐宗、脍、胥敖，南面而不放下心。是什么原因？”舜回答说：“这三个孩子，就像存活在蓬草和艾草之间。如果不放下心，为什么呢？以前十个太阳同时升起，万物都被照亮，更何况德行像太阳一样进步呢！”

齧缺（niè quē）问王倪说：“你知道万物的本质是什么吗？”王倪回答：“我怎么可能知道呢？”“你知道你所不知道的吗？”王倪说：“我怎么可能知道呢！”“那么万物没有智慧吗？”王倪说：“我怎么可能知道呢！虽然，我曾试着说过。庸人难道知道我所谓的知道不就是不知道吗？庸人难道知道我所谓的不知道不就是知道吗？而且我曾试着问你：民众睡觉时就腰痛快死，是这样吗？木头处于位置就发抖恐惧，猨猴也是这样吗？三者谁知道如何正确地处于位置？民众吃草养畜，麋鹿吃荐食，螂蛆喜欢甜带，鸱鸦喜欢年老的鼠，四者谁知道天下的正味？猨猵（yuán yī）狙（jū）以为雌性，麋与鹿交配，?与鱼游。毛嫱丽姬（qī jī），人们所喜欢的，鱼见它深入水中，鸟见它高飞，麋鹿见它决然。四者谁知道天下的正色呢？从我自己观察，仁义的端点，是非的道路，混乱不堪，难道我能知道它的辩别吗！”齧缺说：“你不懂得利害，那至人（zhì rén）当然不懂得利害了吗？”王倪回答说：“至人神奇了！大泽（dà zé）燃烧却不能发热，河汉（hé hàn）流动却不能变冷，急雷破山，飘风振海却不能使之惊动。如果是这样的人，乘云气，骑日月，游荡在四海之外。生死对他而言毫不改变，何况是利害的端点呢！”

瞿鹊子问长梧子说：“我听闻诸位夫子，圣人不从事于务，不追求利益，不违背害处，不喜求，不依靠道；无谓有谓，有谓无谓，而游荡在尘垢之外。夫子以为是轻浮的言论，而我认为是妙道的实践。你觉得怎么样？”长梧子回答说：“这是黄帝所听到的灵光（荧）啊，而丘（qiū）有什么资格知道呢？而且你也早早地计算，看到鸡蛋就求得夜晚的时间，看到弹拨就求得鸮（xiāo）烧烤。以前我曾为你妄言，你却妄听了。怎样靠近日月，携带宇宙，使其吻合，置于滑涽（huáng）中，以统治相尊。众人忙碌不已，圣人愚昧，历经万岁却一成不变。万物自然而然地相互蕴藏。”

我怎么能知道生的不是迷惑吗？我怎么能知道死的不是虚弱的丧失，不知道归宿吗！丽姬（lì jī），艾封人之子。晋国开始得到了它，涕泣沾襟；后来到了王那里，与王同睡在筐床上，吃草养畜，后来后悔自己的泣泣。怎么能知道死的人不会后悔自己刚开始的生呢！梦见喝酒的人，早上就哭泣；梦见哭泣的人，早上就去田猎。梦的时候，不知道是梦。梦中又解读梦，醒来后才知道是梦。而且有大觉悟后才知道这是大梦，愚者自以为觉悟，窃窃然知道。君主啊，牧养啊，固然如此！丘（qiū）和你，都是梦啊；我说你在做梦，也是梦啊。这种言论，叫做吊诡。万世之后遇到一个大圣，知道他解悟的人，是早晚相遇的。

即使我和若（ruò）辩论，如果若胜我，我就不是若；如果果真如此，我就不是。如果若不胜我，我就是真，也不是。他要么是，要么不是。他们既是，也不是。他们都是真的，也都不是真的。我和若不能相互了解，那人们就固然昏暗。我是谁来纠正呢？让同于若的人来纠正呢？既然我和若相同，怎么能纠正呢！让同于我的人来纠正呢？既然我和若相同，怎么能纠正呢！让不同于我和若的人来纠正呢？既然我和若不同，怎么能纠正呢！让同于我和若的人来纠正呢？既然我和若相同，怎么能纠正呢！然而，我和若以及人们都无法相互了解，只是等待彼此吗？

什么是用天的规律来调和？回答说：不是是，也不是不是。如果是就和不是的区别没有辩论；如果是自然的，就和不然的区别也没有辩论。声音的相互作用，好像它们不相互作用。用天的规律来调和，依靠漫延（màn yǎn），所以穷尽年岁。忘记年岁，忘记义理，振动于无尽，因此寄寓于无尽之中。

罔两问景说：“以前你行走，现在你停下；以前你坐着，现在你起来；怎么会没有特别的行为呢？”景回答说：“我有依靠然后如此吗？我所依靠的又有依靠然后如此吗？我依靠蛇蚹蜩翼（tiāo yì）吗？不识所以然！不识所以不然！”

昔日庄周梦为蝴蝶，栩栩然蝴蝶，自我认为适意了！不知道庄周是谁。忽然醒来，就又是庄周。不知道庄周是梦见蝴蝶，还是蝴蝶梦见庄周？庄周和蝴蝶，就必然有分别了。这就是物化的意思。

# 典故解释

1. **南郭子綦**：南郭子綦是庄子记载的一个隐士形象，象征着隐居避世的人。
2. **偃（yǎn）**：在古代指安静隐居，这里指放下心中的纷扰，追求内心的宁静。
3. **神禹（yǔ）**：指传说中的大禹，是中国古代著名的治水英雄，象征智慧和治国能力。
4. **黄帝**：中国古代传说中的一位帝王，象征智慧和统治者的典范。
5. **庄周梦为蝴蝶**：庄子所著的著名寓言故事，讲述庄子梦见自己变成蝴蝶，醒来后不知是庄子梦见变成蝴蝶，还是蝴蝶梦见变成庄子，象征“物我两忘”的哲学思想。
6. **狙公**：传说中的人物，具体身份不详，这里用来说明圣人不参与世俗的争论。
7. **昭氏**：指古代家族中的贤人或学者，这里象征技艺精湛但未能理解道理的人。
8. **麋鹿**：古代传说中的神兽，象征高洁与独立。

# 深刻句子汇总

1. “非彼无我，非我无所取。是亦近矣，而不知其所为使。”
   意思是：不是彼此没有我，也不是我没有取舍。这已经很接近了，却不知道它是如何运作的。
2. “昔者庄周梦为蝴蝶，栩栩然蝴蝶也，自喻适志与！不知周也。俄然觉，则蘧蘧然周也。不知周之梦为蝴蝶与，蝴蝶之梦为周与？周与蝴蝶，则必有分矣。此之谓物化。”
   意思是：庄周曾梦见自己变成蝴蝶，完全陶醉其中，却不知自己是庄周。突然醒来，又是庄周。他不知道是庄周梦见蝴蝶，还是蝴蝶梦见庄周，这说明了物化的变化。
3. “道未始有封，言未始有常，为是而有畛也。”
   意思是：道从未有过固定的界限，言语从未有过常规，用“是”来界定才有了界限。
4. “喜怒哀乐，虑叹变埶心，姚佚启态；乐出虚，蒸成菌。”
   意思是：喜怒哀乐，忧虑叹息改变了心情，放纵欲望表现出各种状态；快乐源于虚无，滋养出了菌类。
5. “万物无成与毁，复通为一。唯达者知通为一，为是不用而寓诸庸。”
   意思是：万物没有成就与毁灭，重新归于一体。只有通晓道理的人知道归于一体，成为“是”却不用，寄托于庸俗中。

# 章节总结

本章通过庄子与子綦、子游等人物的对话，探讨了道与万物的本质及其变化，强调了“齐物论”的核心思想——万物平等，无有高下之分。庄子认为，人们常被名利、是非所困扰，无法理解事物的本源和自然规律。他通过隐喻和寓言，揭示了道的无形无象、万物相互依存的真理，提倡超越对立，追求内心的自由与宁静。
