# Workbench

Integrated development environment with terminal, Git operations, file browser, and port proxy.

![Workbench](/images/workbench.jpg)

## Features

- **Terminal** — Full terminal emulator (xterm.js) with PTY support, run any command on the agent machine
- **Git** — Status viewer, diff viewer, branch management, all within the browser
- **File browser** — Browse and edit files on the agent machine with CodeMirror editor
- **Port proxy** — Forward agent local ports to your browser for previewing web apps
