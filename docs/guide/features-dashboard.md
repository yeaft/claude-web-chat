# Admin Dashboard

Usage statistics and system monitoring for administrators.

![Dashboard](/images/dashboard.jpg)

## Features

- **User activity metrics** — Track usage with time-based filtering (today/week/month)
- **Per-user breakdown** — Detailed stats including messages, sessions, requests, and traffic
- **Agent monitoring** — Connected agent status and latency monitoring
- **Mobile-responsive** — Card layout adapts to all screen sizes
